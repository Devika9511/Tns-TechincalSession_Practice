# AI & ML Learning Tasks – TNS India Foundation

## Overview

This repository contains my practice tasks, exercises, and learning notebooks completed as part of the **Artificial Intelligence & Machine Learning Training Program** conducted by **TNS India Foundation**.

The primary goal of this repository is to document my learning journey, strengthen my Python programming skills, and build a strong foundation for AI and ML concepts through hands-on practice.

## Contents

The notebooks in this repository cover:

* Python Data Types
* Strings and String Methods
* Functions
* Conditional Statements
* Loops
* Prime Numbers
* Factorial Programs
* Palindrome and Armstrong Numbers
* Pattern Printing
* Stack
* Queue
* Linked List
* Trees and Tree Traversals
* Problem Solving Exercises

## Learning Objectives

* Understand Python programming fundamentals.
* Practice writing clean and efficient code.
* Learn core data structures and their applications.
* Build a strong foundation for Artificial Intelligence and Machine Learning.
* Develop logical thinking and problem-solving skills.
